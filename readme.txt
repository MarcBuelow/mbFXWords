This is a NetBeans project with Ant build script.
To just execute mbFXWords without compiling:

'dist' directory contains complete portable application.

In Windows double click on mbFXWords.jar. Make sure JRE 1.8 is installed.

In Linux move 'dist' to your home directory, make mbFXWords.jar executable
(Console: sudo chmod +x mbFXWord.jar) and start (Console: java -jar mbFXWord.jar). Make sure JRE 1.8 is installed.
