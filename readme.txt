For developers:

This is a NetBeans project with Ant build script.


To just execute mbFXWords without compiling:


'dist' directory contains portable application.


Make sure JRE 1.8 is installed
.

In Windows:
- execute dist-make-runnable.bat
- double click on mbFXWords.jar in dist directory.


In Linux:
- execute dist-make-runnable.sh
- move 'dist' to your home directory
- make mbFXWords.jar executable 
(Console: sudo chmod +x mbFXWord.jar)
- start (Console: java -jar mbFXWord.jar)

Alternatively use 'mbFXWords.exe' for Windows
or 'mbFXWords.sh' for Linux.
